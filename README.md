# BuscoBrete

## Plataforma web para conectar PYMES con talento local en Costa Rica

**BuscoBrete** es una aplicación web desarrollada para el curso **SC-502 Ambiente Web Cliente/Servidor** de la **Universidad Fidélitas**.  
Su propósito es conectar pequeñas y medianas empresas con personas en búsqueda de empleo en Costa Rica, priorizando oportunidades laborales en zonas locales y facilitando un proceso de búsqueda más claro, accesible y ordenado.

---

## Descripción del proyecto

El proyecto surge como una propuesta orientada a resolver una necesidad real: mejorar la conexión entre empleadores locales y personas que buscan oportunidades de trabajo dentro de sus comunidades.

La solución plantea una aplicación web con diferentes tipos de usuario, funcionalidades específicas por rol y una estructura enfocada en la experiencia del usuario, la organización de la información y la centralización de datos.

---

## Objetivo general

Desarrollar una aplicación web llamada **BuscoBrete** que permita a PYMES postular posiciones disponibles en sus zonas locales para ayudar al emprendimiento en las regiones del país.

---

## Objetivos específicos

- Implementar al proyecto un diseño intuitivo basado en los fundamentos de **UX/UI** que facilite la experiencia de buscar un trabajo.
- Aplicar tecnologías web tales como **HTML5, PHP y MySQL** para implementar las funcionalidades básicas según los roles definidos.
- Facilitar la conexión entre empleadores y personas en búsqueda de empleo en Costa Rica, priorizando las zonas locales.
- Almacenar y administrar de forma centralizada y autónoma la información de los usuarios finales que participen en la aplicación web.

---

## Roles del sistema

La aplicación contempla tres roles principales:

- **Administrador**
- **Reclutador**
- **Usuario**
